# Xiaobi — AI-Powered Storytelling Panda 🐼✨

Xiaobi is a personalized children's story generator that creates unique bedtime stories with AI-narrated audio and beautiful illustrations. Tell Xiaobi about your child — their name, interests, and dreams — and he'll weave a magical story just for them.

## Features

- **Conversational Story Creation** — Chat with Xiaobi to build a personalized story context (child's name, age, interests, themes, moral lessons)
- **AI Story Generation** — Powered by Google Gemini 3.1 Pro to craft rich, multi-scene narratives
- **Voice Narration** — Professional-quality narration via ElevenLabs text-to-speech with multiple voice options
- **AI Illustrations** — Every scene gets a stunning illustration generated by OpenAI DALL-E 3
- **Interactive Playback** — Full-screen story player with audio sync, image crossfade transitions, play/pause, skip, volume, speed controls, and captions
- **Slideshow Mode** — Manual forward/back navigation with visible transcripts and no audio
- **Surprise Me!** — One-click random story generation for instant fun
- **Unchained Mode 🔥** — Mature story themes for adult entertainment (with content-policy-safe image fallbacks)
- **Talk to Xiaobi** — Video chat with an AI panda narrator about your story (powered by Tavus CVI)
- **Story Library** — Save, replay, and favorite your stories with playback mode toggle

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite + TypeScript + Tailwind CSS |
| UI Components | shadcn/ui + Radix primitives |
| Backend | Lovable Cloud (Supabase — Postgres, Auth, Edge Functions, Storage) |
| LLM | Google Gemini 3.1 Pro via Lovable AI Gateway |
| Voice | ElevenLabs (Turbo v2.5, streaming) |
| Images | OpenAI DALL-E 3 |
| Video Avatar | Tavus CVI API (WebRTC) |

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (`.env` is auto-configured via Lovable Cloud)
4. Run dev server: `npm run dev`

## Design

Warm, child-friendly aesthetic with navy (#1B2D4F), soft gold (#F4C753), cloud white (#FAFBFC), and coral (#FF8A80). Nunito for headings, Inter for body text, 16px border radius throughout.
